## Operational Principles

- **Least Privilege by Default** ‚ÄĒ Every process, user, and service runs with the minimum access required.
- **Air-Gap Awareness** ‚ÄĒ Core tooling is designed to function without internet connectivity.
- **Reproducibility** ‚ÄĒ Builds are deterministic; any deployment can be validated against a known-good state.
- **Separation of Concerns** ‚ÄĒ Red-team tooling is containerized and isolated from defensive instrumentation.
- **No Telemetry** ‚ÄĒ No data leaves the system without explicit operator action.
