## 📖 Background

Steganography differs from cryptography: while cryptography makes data unreadable, steganography makes data invisible. An encrypted message is obviously secret; a steganographic message looks like an ordinary file. Covert channels extend this concept to network protocols — DNS queries, ICMP pings, and HTTP headers can all carry hidden payloads.

Key concepts:
- **LSB (Least Significant Bit) embedding** — Replacing the least significant bits of pixel data with hidden message bits
- **Statistical steganalysis** — Detecting hidden data through statistical anomalies in carrier files
- **DNS tunneling** — Encoding data in DNS query/response fields to bypass firewalls
- **ICMP tunneling** — Hiding data in ICMP echo request/reply payloads
- **Cover object** — The innocent-looking file that carries the hidden payload

### Phase 3: DNS Tunneling — Covert Network Channels

1. Understand DNS tunneling architecture:
   ```
   Client → DNS Query (data encoded in subdomain) → DNS Server
   DNS Server → DNS Response (data encoded in TXT record) → Client
   Example: dGhpcyBpcyBzZWNyZXQ.tunnel.example.com
   ```
2. Set up a local DNS tunneling demonstration using dns2tcp:
   ```bash
   sudo apt-get install -y dns2tcp
   ```
3. Simulate DNS tunneling with a Python proof-of-concept:
   ```python
   import base64

   def encode_dns_query(data, domain="tunnel.example.com"):
       """Encode data as a DNS-safe subdomain label."""
       encoded = base64.b32encode(data.encode()).decode().rstrip("=").lower()
       # DNS labels max 63 chars — split if needed
       labels = [encoded[i:i+63] for i in range(0, len(encoded), 63)]
       return ".".join(labels) + "." + domain

   def decode_dns_query(query, domain="tunnel.example.com"):
       """Decode data from a DNS query subdomain."""
       encoded = query.replace("." + domain, "").replace(".", "")
       padding = "=" * (8 - len(encoded) % 8) if len(encoded) % 8 else ""
       return base64.b32decode(encoded.upper() + padding).decode()

   # Demonstration
   secret = "GRIMOIRE{dns_tunnel_detected}"
   query = encode_dns_query(secret)
   print(f"DNS Query: {query}")
   recovered = decode_dns_query(query)
   print(f"Recovered: {recovered}")
   ```
4. Build detection rules for DNS tunneling:
   - Unusually long subdomain labels (>30 characters)
   - High entropy in DNS query names
   - Abnormal volume of TXT record queries to a single domain
5. Write a detection script that flags suspicious DNS queries:
   ```bash
   # Analyze DNS logs for tunneling indicators
   awk '{print length($1), $1}' dns_queries.log | sort -rn | head -20
   ```

### Phase 4: ICMP Covert Channels

1. Understand how ICMP tunneling works:

   ```bash
   sudo tcpdump -i lo icmp -c 20 -w ~/grimoire/lab-033/icmp_capture.pcap &
   ```

   ```bash
   tcpdump -r ~/grimoire/lab-033/icmp_capture.pcap -X | grep -A 2 "GRIMOIRE"
   ```
4. Document detection signatures: unusual payload sizes, non-standard padding patterns, high ICMP traffic volume

## 🛠️ Build Your Own

**Build Objective:** Create a custom steganography toolkit that can embed and extract hidden data in images, with detection capabilities.

### Requirements
- Embed arbitrary text messages in PNG/JPEG images using LSB encoding
- Extract hidden messages from steganographic images
- Detect whether an image likely contains hidden data via statistical analysis
- Support password-based encryption of the hidden payload
- Provide a CLI interface for all operations

### Starter Framework
```python
#!/usr/bin/env python3
"""GRIMOIRE Steganography Toolkit - LAB-033 Build Your Own"""

import argparse
import hashlib
from PIL import Image
import numpy as np


class StegToolkit:
    def __init__(self, password=None):
        self.password = password

    def _xor_encrypt(self, data, key):
        """Simple XOR encryption for the hidden payload."""
        key_hash = hashlib.sha256(key.encode()).digest()
        return bytes([b ^ key_hash[i % len(key_hash)] for i, b in enumerate(data)])

    def embed(self, carrier_path, message, output_path):
        """Embed a message into an image using LSB encoding."""
        img = Image.open(carrier_path).convert("RGB")
        pixels = np.array(img)

        # Convert message to binary
        msg_bytes = message.encode("utf-8")
        if self.password:
            msg_bytes = self._xor_encrypt(msg_bytes, self.password)

        # Prepend message length (4 bytes, big-endian)
        length_prefix = len(msg_bytes).to_bytes(4, byteorder="big")
        payload = length_prefix + msg_bytes
        bits = ''.join(format(b, '08b') for b in payload)

        # TODO: Check if image has enough capacity
        flat = pixels.flatten()
        if len(bits) > len(flat):
            raise ValueError("Message too large for carrier image")

        # TODO: Embed bits into LSB of pixel values
        for i, bit in enumerate(bits):
            flat[i] = (flat[i] & 0xFE) | int(bit)

        stego = flat.reshape(pixels.shape)
        Image.fromarray(stego.astype(np.uint8)).save(output_path)
        print(f"[+] Message embedded in {output_path}")

    def extract(self, stego_path):
        """Extract a hidden message from a steganographic image."""
        img = Image.open(stego_path).convert("RGB")
        flat = np.array(img).flatten()

        # TODO: Read length prefix (first 32 bits)
        length_bits = ''.join(str(flat[i] & 1) for i in range(32))
        msg_length = int(length_bits, 2)

        # TODO: Read message bits
        msg_bits = ''.join(str(flat[i] & 1) for i in range(32, 32 + msg_length * 8))
        msg_bytes = bytes(int(msg_bits[i:i+8], 2) for i in range(0, len(msg_bits), 8))

        if self.password:
            msg_bytes = self._xor_encrypt(msg_bytes, self.password)

        return msg_bytes.decode("utf-8")

    def analyze(self, image_path):
        """Analyze an image for steganographic indicators."""
        img = Image.open(image_path).convert("RGB")
        pixels = np.array(img)

        # TODO: Compute LSB statistics per channel
        results = {}
        channels = ["Red", "Green", "Blue"]
        for i, channel in enumerate(channels):
            lsb = pixels[:,:,i] & 1
            ratio = np.mean(lsb)
            results[channel] = {
                "lsb_ratio": round(float(ratio), 4),
                "suspicious": abs(ratio - 0.5) < 0.01
            }

        # TODO: Chi-squared analysis for more robust detection
        print(f"[*] LSB Analysis for {image_path}:")
        for ch, data in results.items():
            status = "⚠ SUSPICIOUS" if data["suspicious"] else "✓ Clean"
            print(f"    {ch}: ratio={data['lsb_ratio']} {status}")
        return results


if __name__ == "__main__":
    parser = argparse.ArgumentParser(description="GRIMOIRE Steganography Toolkit")
    sub = parser.add_subparsers(dest="command")

    embed_p = sub.add_parser("embed", help="Embed a message in an image")
    embed_p.add_argument("carrier", help="Carrier image path")
    embed_p.add_argument("message", help="Message to embed")
    embed_p.add_argument("-o", "--output", default="stego_output.png")
    embed_p.add_argument("-p", "--password", help="Encryption password")

    extract_p = sub.add_parser("extract", help="Extract hidden message")
    extract_p.add_argument("image", help="Steganographic image path")
    extract_p.add_argument("-p", "--password", help="Decryption password")

    analyze_p = sub.add_parser("analyze", help="Analyze image for hidden data")
    analyze_p.add_argument("image", help="Image to analyze")

    args = parser.parse_args()
    toolkit = StegToolkit(password=getattr(args, "password", None))

    if args.command == "embed":
        toolkit.embed(args.carrier, args.message, args.output)
    elif args.command == "extract":
        print(toolkit.extract(args.image))
    elif args.command == "analyze":
        toolkit.analyze(args.image)
```
