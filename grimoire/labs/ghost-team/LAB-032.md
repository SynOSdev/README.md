## 📖 Background

The dark web refers to overlay networks (primarily Tor) that require special software to access. While often sensationalized, it is a vital source of threat intelligence for defenders. Data breaches are frequently posted to paste sites, stolen credentials are traded in forums, and threat actors discuss TTPs openly.

Key concepts:
- **Onion routing** — Traffic encrypted in layers and relayed through multiple nodes to obscure origin
- **Hidden services (.onion)** — Services only accessible through the Tor network
- **Paste sites** — Anonymous text-sharing platforms where leaks and dumps are published
- **Threat intelligence feeds** — Automated streams of indicators sourced from dark web monitoring
- **Safe research practices** — Isolating your research environment to prevent exposure

## 🔬 Lab Instructions

### Phase 1: Understanding Tor Architecture

1. Install Tor and torsocks in your lab environment:
   ```bash
   sudo apt-get install -y tor torsocks
   ```
2. Start the Tor service and verify connectivity:
   ```bash
   sudo systemctl start tor
   torsocks curl -s https://check.torproject.org/api/ip
   ```
3. Examine the Tor configuration at `/etc/tor/torrc`:
   - Identify entry guards, circuit timeout, and SOCKS port settings
4. Trace a Tor circuit to understand the relay chain:
   ```bash
   torsocks curl -s https://check.torproject.org | grep -i "congratulations"
   ```
5. Review how onion routing provides anonymity:
   - Client → Guard Node → Middle Relay → Exit Node → Destination
   - For .onion: Client → Guard → Middle → Rendezvous Point ← Hidden Service

### Phase 2: Safe Research Environment Setup

1. Configure a dedicated research profile with no personal data:
   ```bash
   mkdir -p ~/grimoire/lab-032/research_env
   echo "NoPersonalData=true" > ~/grimoire/lab-032/research_env/config.txt
   ```
2. Set up torsocks for Python-based requests:
   ```bash
   pip install requests[socks] PySocks
   ```
3. Test programmatic Tor connectivity:
   ```python
   import requests
   proxies = {"http": "socks5h://127.0.0.1:9050",
              "https": "socks5h://127.0.0.1:9050"}
   r = requests.get("https://check.torproject.org/api/ip", proxies=proxies, timeout=30)
   print(r.json())
   ```
4. Establish research ground rules:
   - Never log in to any service with real credentials
   - Never download unknown executables
   - Document everything with timestamps

### Phase 3: Paste Site Monitoring for Threat Indicators

1. Set up a keyword watchlist for monitoring:

3. Parse and filter results programmatically:
   ```python
   import json
   ```

4. Log all alerts with timestamps and source references

## 🛠️ Build Your Own

**Build Objective:** Create a custom dark web monitoring alert tool that checks for specified keywords and indicators in public paste sites and produces structured alerts.

### Requirements
- Accept a watchlist of keywords, domains, and email patterns
- Poll public paste-like APIs on a configurable schedule
- Match content against watchlist using exact and fuzzy matching
- Generate timestamped alerts with severity classification
- Export alerts as JSON and optional email/webhook notification

### Starter Framework
```python
#!/usr/bin/env python3
"""GRIMOIRE Dark Web Monitor - LAB-032 Build Your Own"""

import json
import time
import re
import argparse
from datetime import datetime


class DarkWebMonitor:
    def __init__(self, watchlist_path, interval=300):
        self.watchlist = self._load_watchlist(watchlist_path)
        self.interval = interval
        self.alerts = []

    def _load_watchlist(self, path):
        """Load keywords and patterns from watchlist file."""
        keywords = []
        with open(path) as f:
            for line in f:
                term = line.strip()
                if term and not term.startswith("#"):
                    keywords.append(term.lower())
        return keywords

    def fetch_pastes(self):
        """Fetch recent pastes from public sources.
        TODO: Implement fetching from real public paste APIs.
        Returns simulated pastes for lab environment."""
        return [
            {"id": f"paste-{int(time.time())}-{i}",
             "timestamp": datetime.utcnow().isoformat(),
             "content": f"Sample paste content {i}",
             "source": "lab-simulation"}
            for i in range(5)
        ]

    def scan_paste(self, paste):
        """Scan a single paste against the watchlist."""
        matches = []
        content_lower = paste.get("content", "").lower()
        for keyword in self.watchlist:
            if keyword in content_lower:
                matches.append({
                    "keyword": keyword,
                    "paste_id": paste["id"],
                    "source": paste.get("source", "unknown"),
                    "timestamp": datetime.utcnow().isoformat(),
                    "severity": self._classify_severity(keyword),
                    "snippet": content_lower[:200]
                })
        return matches

    def _classify_severity(self, keyword):
        """Classify alert severity based on keyword category."""
        # TODO: Implement severity rules — credentials = high,
        #       brand mentions = medium, general keywords = low
        high_indicators = ["password", "credential", "database dump", "ssn", "credit card"]
        for indicator in high_indicators:
            if indicator in keyword:
                return "HIGH"
        return "MEDIUM"

    def run_scan_cycle(self):
        """Execute one monitoring cycle."""
        pastes = self.fetch_pastes()
        for paste in pastes:
            matches = self.scan_paste(paste)
            self.alerts.extend(matches)
        return len(self.alerts)

    def monitor(self):
        """Run continuous monitoring loop."""
        print(f"[*] Dark Web Monitor started — watching {len(self.watchlist)} keywords")
        try:
            while True:
                count = self.run_scan_cycle()
                if count > 0:
                    print(f"[!] {count} total alerts generated")
                time.sleep(self.interval)
        except KeyboardInterrupt:
            self.export_alerts()

    def export_alerts(self, path="dwm_alerts.json"):
        """Export all alerts to JSON."""
        with open(path, "w") as f:
            json.dump({"alerts": self.alerts, "total": len(self.alerts)}, f, indent=2)
        print(f"[+] {len(self.alerts)} alerts exported to {path}")


if __name__ == "__main__":
    parser = argparse.ArgumentParser(description="GRIMOIRE Dark Web Monitor")
    parser.add_argument("watchlist", help="Path to watchlist file")
    parser.add_argument("-i", "--interval", type=int, default=300,
                        help="Polling interval in seconds")
    parser.add_argument("-o", "--output", default="dwm_alerts.json",
                        help="Alert output path")
    args = parser.parse_args()

    monitor = DarkWebMonitor(args.watchlist, args.interval)
    monitor.run_scan_cycle()
    monitor.export_alerts(args.output)
```
