## 📖 Background

Attribution — identifying who is behind a cyberattack — is one of the most challenging problems in cybersecurity. Technical indicators alone are rarely sufficient; adversaries reuse tools, share infrastructure, and plant false flags. Rigorous attribution requires multiple independent evidence streams analyzed through structured frameworks.

Key concepts:
- **Diamond Model** — Analyzes intrusions through four vertices: Adversary, Capability, Infrastructure, Victim
- **MITRE ATT&CK** — A knowledge base of adversary TTPs organized by tactics (what they want to achieve) and techniques (how they do it)
- **Kill Chain** — Phases of an intrusion: Reconnaissance → Weaponization → Delivery → Exploitation → Installation → C2 → Actions
- **Campaign tracking** — Linking related intrusions over time through shared indicators and TTPs
- **Confidence levels** — Rating attribution certainty: Almost Certain, Probable, Possible, Unlikely

### Phase 1: The Diamond Model of Intrusion Analysis

1. Understand the four vertices of the Diamond Model:
   ```
   Diamond Model Vertices:
   ┌─────────────────────────┐
   │       Adversary          │ ← Who: threat actor or group
   │           │              │
   │    Infrastructure ───── Capability │ ← How: tools, malware, exploits
   │           │              │
   │        Victim            │ ← Who was targeted
   └─────────────────────────┘
   Meta-Features: Timestamp, Phase, Result, Direction, Methodology, Resources
   ```
2. Create a Diamond Model event from a simulated intrusion:
   ```python
   diamond_event = {
       "event_id": "GRIM-2049-001",
       "timestamp": "2049-03-15T14:30:00Z",
       "adversary": {
           "name": "Unknown — assess as state-sponsored",
           "aliases": [],
           "motivation": "Espionage"
       },
       "capability": {
           "tools": ["Custom RAT", "Mimikatz", "PsExec"],
           "techniques": ["T1566.001", "T1053.005", "T1003.001"],
           "sophistication": "High"
       },
       "infrastructure": {
           "c2_domains": ["update.fictional-cdn.example.com"],

       },
       "victim": {
           "sector": "Defense Industrial Base",
           "region": "North America",
   ```

   ```python
   def generate_navigator_layer(ttp_mapping, actor_name="Unknown Actor"):
       """Generate a MITRE ATT&CK Navigator JSON layer."""
       techniques = []
       for tactic, techs in ttp_mapping.items():
           for t in techs:
               techniques.append({
                   "techniqueID": t["technique"],
                   "tactic": tactic.lower().replace(" ", "-"),
                   "color": "#ff6666",
                   "comment": t.get("evidence", ""),
                   "enabled": True
               })
       layer = {
           "name": f"{actor_name} TTP Coverage",
           "versions": {"attack": "14", "navigator": "4.9", "layer": "4.5"},
           "domain": "enterprise-attack",
           "description": f"Observed TTPs for {actor_name}",
           "techniques": techniques,
           "gradient": {"colors": ["#ffffff", "#ff6666"], "minValue": 0, "maxValue": 1}
       }
       return layer

   layer = generate_navigator_layer(ttp_mapping, "GRIM-ACTOR-001")
   with open("attack_layer.json", "w") as f:
       json.dump(layer, f, indent=2)
   print("[+] ATT&CK Navigator layer saved to attack_layer.json")
   ```

### Phase 3: Attribution Methodology

1. Apply the multi-dimensional attribution framework:
   ```python
   attribution_dimensions = {
       "technical": {
           "weight": 0.3,
           "indicators": [
               {"type": "malware_family", "value": "CustomRAT v2.1",
                "linked_to": "GRIM-ACTOR-001", "confidence": 0.7},
               {"type": "c2_infrastructure", "value": "198.51.100.0/24",
                "linked_to": "GRIM-ACTOR-001", "confidence": 0.6}
           ]
       },
       "behavioral": {
           "weight": 0.3,
           "indicators": [
               {"type": "working_hours", "value": "UTC+8 business hours",
                "linked_to": "East Asia region", "confidence": 0.5},
               {"type": "target_selection", "value": "Defense sector",
                "linked_to": "State-sponsored espionage", "confidence": 0.6}
           ]
       },
       "operational": {
           "weight": 0.2,
           "indicators": [
               {"type": "ttp_overlap", "value": "85% overlap with known group",
                "linked_to": "GRIM-ACTOR-001", "confidence": 0.8},
               {"type": "tooling", "value": "Shared code with previous campaigns",
                "linked_to": "GRIM-ACTOR-001", "confidence": 0.7}
           ]
       },
       "contextual": {
           "weight": 0.2,
           "indicators": [
               {"type": "geopolitical", "value": "Tensions in target region",
                "linked_to": "State motivation", "confidence": 0.4},
               {"type": "timing", "value": "Campaign coincides with policy events",
                "linked_to": "State motivation", "confidence": 0.3}
           ]
       }
   }

   # Calculate weighted attribution confidence
   total_confidence = 0
   for dimension, data in attribution_dimensions.items():
       dim_avg = sum(i["confidence"] for i in data["indicators"]) / len(data["indicators"])
       weighted = dim_avg * data["weight"]
       total_confidence += weighted
       print(f"  [{dimension}] avg={dim_avg:.2f} weighted={weighted:.2f}")

   print(f"\n  Overall Attribution Confidence: {total_confidence:.2f}")
   print(f"  Level: {'Almost Certain' if total_confidence > 0.85 else 'Probable' if total_confidence > 0.65 else 'Possible' if total_confidence > 0.4 else 'Unlikely'}")
   ```
2. Analyze false flag indicators:

   - Tools from one group used by another (shared tooling)

## 🛠️ Build Your Own

**Build Objective:** Create a custom threat actor profiling system that tracks TTPs, infrastructure, and campaigns with structured data storage.

### Requirements
- Store threat actor profiles with aliases, motivations, and capabilities
- Map and query TTPs using MITRE ATT&CK technique IDs
- Track infrastructure (domains, IPs) linked to actors over time
- Link campaigns to actors with confidence scoring
- Generate formatted intelligence reports from stored data

### Starter Framework
```python
#!/usr/bin/env python3
"""GRIMOIRE Threat Actor Profiler - LAB-036 Build Your Own"""

import json
import argparse
from datetime import datetime
from collections import defaultdict


class ThreatActorDB:
    def __init__(self, db_path="threat_actors.json"):
        self.db_path = db_path
        self.data = self._load()

    def _load(self):
        try:
            with open(self.db_path) as f:
                return json.load(f)
        except (FileNotFoundError, json.JSONDecodeError):
            return {"actors": {}, "campaigns": {}, "iocs": []}

    def _save(self):
        with open(self.db_path, "w") as f:
            json.dump(self.data, f, indent=2)

    def add_actor(self, actor_id, name, aliases=None, motivation="Unknown",
                  sophistication="Unknown"):
        """Register a new threat actor."""
        self.data["actors"][actor_id] = {
            "name": name,
            "aliases": aliases or [],
            "motivation": motivation,
            "sophistication": sophistication,
            "ttps": [],
            "infrastructure": [],
            "first_seen": datetime.utcnow().isoformat(),
            "last_updated": datetime.utcnow().isoformat()
        }
        self._save()
        print(f"[+] Actor '{name}' added as {actor_id}")

    def add_ttp(self, actor_id, technique_id, tactic, evidence=""):
        """Link a MITRE ATT&CK TTP to an actor."""
        if actor_id not in self.data["actors"]:
            print(f"[-] Actor {actor_id} not found")
            return
        self.data["actors"][actor_id]["ttps"].append({
            "technique_id": technique_id,
            "tactic": tactic,
            "evidence": evidence,
            "date_observed": datetime.utcnow().isoformat()
        })
        self.data["actors"][actor_id]["last_updated"] = datetime.utcnow().isoformat()
        self._save()
        print(f"[+] TTP {technique_id} added to {actor_id}")

    def add_infrastructure(self, actor_id, indicator_type, value, status="active"):
        """Track infrastructure linked to an actor."""
        if actor_id not in self.data["actors"]:
            print(f"[-] Actor {actor_id} not found")
            return
        self.data["actors"][actor_id]["infrastructure"].append({
            "type": indicator_type,
            "value": value,
            "status": status,
            "first_seen": datetime.utcnow().isoformat()
        })
        self._save()
        print(f"[+] Infrastructure {value} linked to {actor_id}")

    def add_campaign(self, campaign_id, actor_id, name, start_date,
                     targets=None, confidence="Possible"):
        """Register a campaign linked to an actor."""
        self.data["campaigns"][campaign_id] = {
            "name": name,
            "actor_id": actor_id,
            "start_date": start_date,
            "targets": targets or [],
            "confidence": confidence,
            "events": []
        }
        self._save()
        print(f"[+] Campaign '{name}' registered as {campaign_id}")

    def get_actor_profile(self, actor_id):
        """Retrieve full actor profile."""
        actor = self.data["actors"].get(actor_id)
        if not actor:
            print(f"[-] Actor {actor_id} not found")
            return None
        return actor

    def generate_report(self, actor_id):
        """Generate a formatted intelligence report for an actor."""
        actor = self.get_actor_profile(actor_id)
        if not actor:
            return

        # Gather related campaigns
        campaigns = {cid: c for cid, c in self.data["campaigns"].items()
                     if c["actor_id"] == actor_id}

        # Group TTPs by tactic
        ttps_by_tactic = defaultdict(list)
        for ttp in actor.get("ttps", []):
            ttps_by_tactic[ttp["tactic"]].append(ttp)

        report = []
        report.append(f"# Threat Intelligence Brief: {actor['name']}")
        report.append(f"**Generated:** {datetime.utcnow().isoformat()}")
        report.append(f"**Actor ID:** {actor_id}")
        report.append(f"**Aliases:** {', '.join(actor['aliases']) or 'None known'}")
        report.append(f"**Motivation:** {actor['motivation']}")
        report.append(f"**Sophistication:** {actor['sophistication']}")
        report.append("")
        report.append("## TTPs (MITRE ATT&CK)")
        for tactic, techs in ttps_by_tactic.items():
            report.append(f"\n### {tactic}")
            for t in techs:
                report.append(f"- **{t['technique_id']}**: {t.get('evidence', 'N/A')}")
        report.append("")
        report.append("## Infrastructure")
        for infra in actor.get("infrastructure", []):
            report.append(f"- [{infra['type']}] {infra['value']} ({infra['status']})")
        report.append("")
        report.append("## Linked Campaigns")
        for cid, campaign in campaigns.items():
            report.append(f"- **{campaign['name']}** ({cid}) — "
                          f"Confidence: {campaign['confidence']}, "
                          f"Targets: {', '.join(campaign['targets'])}")

        report_text = "\n".join(report)
        output_path = f"intel_brief_{actor_id}.md"
        with open(output_path, "w") as f:
            f.write(report_text)
        print(f"[+] Intelligence brief saved to {output_path}")
        return report_text


if __name__ == "__main__":
    parser = argparse.ArgumentParser(description="GRIMOIRE Threat Actor Profiler")
    sub = parser.add_subparsers(dest="command")

    add_actor_p = sub.add_parser("add-actor", help="Register a new threat actor")
    add_actor_p.add_argument("actor_id", help="Unique actor identifier")
    add_actor_p.add_argument("name", help="Actor name")
    add_actor_p.add_argument("--aliases", nargs="*", default=[])
    add_actor_p.add_argument("--motivation", default="Unknown")
    add_actor_p.add_argument("--sophistication", default="Unknown")

    add_ttp_p = sub.add_parser("add-ttp", help="Add a TTP to an actor")
    add_ttp_p.add_argument("actor_id")
    add_ttp_p.add_argument("technique_id", help="MITRE ATT&CK technique ID")
    add_ttp_p.add_argument("tactic", help="ATT&CK tactic name")
    add_ttp_p.add_argument("--evidence", default="")

    add_infra_p = sub.add_parser("add-infra", help="Add infrastructure to an actor")
    add_infra_p.add_argument("actor_id")
    add_infra_p.add_argument("type", choices=["domain", "ip", "url", "email"])
    add_infra_p.add_argument("value")

    report_p = sub.add_parser("report", help="Generate intelligence brief")
    report_p.add_argument("actor_id")

    profile_p = sub.add_parser("profile", help="View actor profile")
    profile_p.add_argument("actor_id")

    args = parser.parse_args()
    db = ThreatActorDB()

    if args.command == "add-actor":
        db.add_actor(args.actor_id, args.name, args.aliases,
                     args.motivation, args.sophistication)
    elif args.command == "add-ttp":
        db.add_ttp(args.actor_id, args.technique_id, args.tactic, args.evidence)
    elif args.command == "add-infra":
        db.add_infrastructure(args.actor_id, args.type, args.value)
    elif args.command == "report":
        db.generate_report(args.actor_id)
    elif args.command == "profile":
        profile = db.get_actor_profile(args.actor_id)
        if profile:
            print(json.dumps(profile, indent=2))
```
