## 📖 Background

OSINT is intelligence gathered from publicly available sources — social media, public records, DNS registrations, job postings, code repositories, and more. Adversaries routinely use OSINT as the first phase of an attack to map targets, identify employees, discover infrastructure, and craft convincing phishing campaigns.

Key concepts:
- **Passive reconnaissance** — Gathering intelligence without directly interacting with the target
- **Username pivoting** — Using a single username to discover linked accounts across platforms
- **Metadata leakage** — Hidden data embedded in files that reveals authorship, GPS coordinates, software versions
- **Social engineering** — Manipulating people into divulging confidential information
- **Phishing indicators** — Telltale signs that an email, message, or site is a lure

5. Run social-analyzer for cross-platform correlation:
   ```bash

   ```
6. Document findings in a structured format: platform, URL, confidence level

### Phase 2: Metadata Extraction and Analysis

1. Download or create sample image/document files for analysis
2. Extract metadata from an image using exiftool:
   ```bash
   exiftool sample_image.jpg
   ```

4. Extract metadata from a PDF document:
   ```bash
   exiftool sample_document.pdf
   ```
5. Practice stripping metadata to understand defensive measures:
   ```bash
   exiftool -all= -overwrite_original sample_image_copy.jpg
   ```
6. Compare before and after — verify all sensitive fields are removed

### Phase 3: Social Engineering Awareness and Phishing Identification

   - Sender legitimacy (header analysis)
   - URL inspection (hover, decode, WHOIS)
   - Language and social pressure tactics
   - Attachment risk assessment
3. Document your analysis in a structured phishing report

### Phase 4: Building an Intelligence Report

1. Aggregate all findings from Phases 1–3 into a single intelligence report

## 🛠️ Build Your Own

**Build Objective:** Create a custom OSINT aggregation framework that queries multiple data sources and produces structured intelligence reports.

### Requirements
- Accept a target identifier (username, domain, or email) as input
- Query multiple open-source APIs and tools in parallel
- Deduplicate and correlate results across sources
- Output a structured JSON report with confidence scoring
- Include a summary view for quick analysis

### Starter Framework
```python
#!/usr/bin/env python3
"""GRIMOIRE OSINT Aggregator - LAB-031 Build Your Own"""

import json
import subprocess
import argparse
import os
from datetime import datetime
from concurrent.futures import ThreadPoolExecutor, as_completed


class OSINTAggregator:
    def __init__(self, target, target_type="username"):
        self.target = target
        self.target_type = target_type
        self.results = {"target": target, "type": target_type,
                        "timestamp": datetime.utcnow().isoformat(),
                        "sources": [], "summary": {}}

    def run_sherlock(self):
        """Enumerate username across social platforms."""
        # TODO: Run sherlock and parse output
        findings = []
        try:
            result = subprocess.run(
                ["sherlock", self.target, "--print-found"],
                capture_output=True, text=True, timeout=120
            )
            for line in result.stdout.strip().splitlines():
                if line.startswith("http"):
                    findings.append({"url": line.strip(), "confidence": "medium"})
        except (FileNotFoundError, subprocess.TimeoutExpired):
            pass
        return {"source": "sherlock", "findings": findings}

    def run_exiftool(self, filepath):
        """Extract metadata from a file."""
        # TODO: Run exiftool -json and parse output
        try:
            result = subprocess.run(
                ["exiftool", "-json", filepath],
                capture_output=True, text=True, timeout=30
            )
            return {"source": "exiftool", "findings": json.loads(result.stdout)}
        except (FileNotFoundError, subprocess.TimeoutExpired, json.JSONDecodeError):
            return {"source": "exiftool", "findings": []}

    def run_dns_lookup(self):
        """Perform DNS-based reconnaissance on a domain target."""
        # TODO: Query DNS records (A, MX, TXT, NS) for the target
        findings = []
        if self.target_type == "domain":
            for rtype in ["A", "MX", "TXT", "NS"]:
                try:
                    result = subprocess.run(
                        ["dig", "+short", self.target, rtype],
                        capture_output=True, text=True, timeout=15
                    )
                    for line in result.stdout.strip().splitlines():
                        findings.append({"record_type": rtype, "value": line.strip()})
                except (FileNotFoundError, subprocess.TimeoutExpired):
                    pass
        return {"source": "dns", "findings": findings}

    def correlate(self):
        """Deduplicate and score findings across all sources."""
        # TODO: Implement cross-source correlation and confidence scoring
        all_findings = []
        for source in self.results["sources"]:
            all_findings.extend(source.get("findings", []))
        self.results["summary"] = {
            "total_sources": len(self.results["sources"]),
            "total_findings": len(all_findings),
            "collection_time": datetime.utcnow().isoformat()
        }

    def collect(self):
        """Run all collection modules in parallel."""
        modules = [self.run_sherlock, self.run_dns_lookup]
        with ThreadPoolExecutor(max_workers=4) as executor:
            futures = {executor.submit(mod): mod.__name__ for mod in modules}
            for future in as_completed(futures):
                self.results["sources"].append(future.result())
        self.correlate()

    def export_report(self, output_path="osint_report.json"):
        """Export the intelligence report as JSON."""
        with open(output_path, "w") as f:
            json.dump(self.results, f, indent=2)
        print(f"[+] Report saved to {output_path}")


if __name__ == "__main__":
    parser = argparse.ArgumentParser(description="GRIMOIRE OSINT Aggregator")
    parser.add_argument("target", help="Target identifier (username, domain, email)")
    parser.add_argument("-t", "--type", choices=["username", "domain", "email"],
                        default="username", help="Target type")
    parser.add_argument("-o", "--output", default="osint_report.json",
                        help="Output report path")
    args = parser.parse_args()

    aggregator = OSINTAggregator(args.target, args.type)
    aggregator.collect()
    aggregator.export_report(args.output)
```
