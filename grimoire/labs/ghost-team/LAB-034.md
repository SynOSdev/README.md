## 📖 Background

OpSec originated in military intelligence — the process of denying adversaries information about your capabilities and intentions. In cybersecurity, OpSec means controlling what information you leak through your tools, accounts, files, and network traffic. A single metadata slip, a reused username, or an unmasked IP address can unravel an entire operation.

Key concepts:
- **Identity compartmentalization** — Separating personas and activities to prevent cross-correlation
- **Traffic analysis resistance** — Preventing adversaries from deducing activities through network patterns
- **Browser fingerprinting** — Identifying users through unique browser configuration signatures
- **Metadata scrubbing** — Removing hidden identifying data from files before distribution
- **Secure communications** — End-to-end encrypted messaging with forward secrecy

## 🔬 Lab Instructions

### Phase 1: Identity Compartmentalization

1. Create separate operational personas for different activities:
   ```bash
   mkdir -p ~/grimoire/lab-034/{persona_research,persona_ops,persona_public}
   ```
2. Define isolation rules for each persona:
   ```bash
   cat > ~/grimoire/lab-034/persona_research/rules.txt << 'EOF'
   - Use only Tor for network access
   - Never cross-reference with real identity
   - Dedicated email created over Tor
   - No reuse of usernames, passwords, or writing style
   - Separate browser profile with no extensions
   EOF
   ```
3. Audit your current digital footprint for cross-persona leaks:
   ```bash
   # Check for identity leaks in git configuration
   git config --global --list 2>/dev/null | grep -E "(user\.|email)"
   # Check for SSH keys that could identify you
   ls -la ~/.ssh/ 2>/dev/null
   # Check shell history for sensitive commands
   wc -l ~/.bash_history 2>/dev/null
   ```
4. Document what information each persona can safely access

### Phase 2: Anonymous Browsing and Traffic Analysis Resistance

1. Verify Tor connectivity and circuit information:
   ```bash
   sudo systemctl start tor
   torsocks curl -s https://check.torproject.org/api/ip | python3 -m json.tool
   ```
2. Test for DNS leaks outside the Tor tunnel:
   ```bash
   # All DNS should resolve through Tor — test for leaks
   torsocks nslookup example.com 2>&1
   # Compare with direct resolution to detect leaks
   ```
3. Understand browser fingerprinting vectors:
   ```python
   # Common fingerprinting attributes that identify you
   fingerprint_vectors = {
       "User-Agent": "Browser, OS, version info",
       "Screen Resolution": "Display size and color depth",
       "Installed Fonts": "Unique font list creates a signature",
       "Canvas Fingerprint": "GPU rendering produces unique hash",
       "WebGL Renderer": "Graphics hardware identification",
       "Timezone": "Geographic location indicator",
       "Language": "Locale and language preferences",
       "Plugins": "Browser plugin list and versions",
       "Do Not Track": "Ironically, enabling this makes you more unique"
   }
   for vector, risk in fingerprint_vectors.items():
       print(f"  [{vector}] → {risk}")
   ```
4. Configure hardened browser settings to minimize fingerprinting:
   ```bash
   cat > ~/grimoire/lab-034/browser_hardening.txt << 'EOF'
   # Firefox about:config hardening
   privacy.resistFingerprinting = true
   privacy.trackingprotection.enabled = true
   media.peerconnection.enabled = false  # Disable WebRTC IP leak
   geo.enabled = false
   dom.battery.enabled = false
   EOF
   ```

### Phase 3: Metadata Scrubbing

1. Install metadata removal tools:
   ```bash
   sudo apt-get install -y mat2 exiftool
   ```
2. Create test files with embedded metadata:
   ```bash
   # Create a document with metadata
   echo "Sensitive report content" > ~/grimoire/lab-034/report.txt
   # Inspect metadata on any available files
   exiftool ~/grimoire/lab-034/report.txt
   ```
3. Scrub metadata from images:
   ```bash
   # View all metadata
   exiftool sample_photo.jpg
   # Remove all metadata
   mat2 sample_photo.jpg
   # Verify removal
   exiftool sample_photo.cleaned.jpg
   ```
4. Scrub metadata from documents:
   ```bash
   # Remove metadata from PDFs
   mat2 document.pdf
   # Verify with exiftool
   exiftool document.cleaned.pdf
   ```
5. Build a batch metadata scrubber:
   ```bash
   #!/bin/bash
   # Scrub all files in a directory
   TARGET_DIR="${1:-.}"
   find "$TARGET_DIR" -type f \( -name "*.jpg" -o -name "*.png" -o -name "*.pdf" \) \
     -exec sh -c 'echo "[*] Scrubbing: $1"; mat2 "$1"' _ {} \;
   echo "[+] Metadata scrubbing complete"
   ```

### Phase 4: Secure Communications

1. Generate a GPG keypair for encrypted communications:
   ```bash
   gpg --batch --gen-key << 'EOF'
   %no-protection
   Key-Type: RSA
   Key-Length: 4096
   Name-Real: Lab Operator
   Name-Email: operator@grimoire.lab
   Expire-Date: 1d
   %commit
   EOF
   ```
2. Encrypt a message for a recipient:
   ```bash
   echo "Classified: Operation successful" | gpg --armor --encrypt -r operator@grimoire.lab
   ```
3. Sign and verify messages for authenticity:
   ```bash
   echo "Verified communication" | gpg --clearsign
   ```
4. Clean up lab keys after exercise:
   ```bash
   gpg --delete-secret-and-public-key operator@grimoire.lab 2>/dev/null
   ```

## 🛠️ Build Your Own

**Build Objective:** Create a custom OpSec audit tool that checks your system for identity leaks and metadata exposure.

### Requirements
- Scan system for identity-revealing configurations (git, SSH, shell history)
- Check network configuration for IP/DNS leak indicators
- Audit files in a target directory for metadata exposure
- Generate a risk score and remediation recommendations
- Output a structured audit report

### Starter Framework
```python
#!/usr/bin/env python3
"""GRIMOIRE OpSec Auditor - LAB-034 Build Your Own"""

import os
import subprocess
import json
import glob
from datetime import datetime
from pathlib import Path


class OpSecAuditor:
    def __init__(self):
        self.findings = []
        self.risk_score = 0

    def _add_finding(self, category, severity, description, remediation):
        """Record an audit finding."""
        weight = {"CRITICAL": 25, "HIGH": 15, "MEDIUM": 10, "LOW": 5}
        self.findings.append({
            "category": category,
            "severity": severity,
            "description": description,
            "remediation": remediation
        })
        self.risk_score += weight.get(severity, 0)

    def audit_git_config(self):
        """Check git configuration for identity leaks."""
        try:
            result = subprocess.run(
                ["git", "config", "--global", "--list"],
                capture_output=True, text=True
            )
            for line in result.stdout.splitlines():
                if "user.email" in line and "example" not in line:
                    self._add_finding(
                        "Identity", "HIGH",
                        f"Real email in git config: {line}",
                        "Use a pseudonymous email: git config --global user.email 'anon@example.com'"
                    )
                if "user.name" in line:
                    self._add_finding(
                        "Identity", "MEDIUM",
                        f"Name in git config: {line}",
                        "Use a pseudonym: git config --global user.name 'operator'"
                    )
        except FileNotFoundError:
            pass

    def audit_ssh_keys(self):
        """Check for SSH keys that could fingerprint the user."""
        ssh_dir = Path.home() / ".ssh"
        if ssh_dir.exists():
            pub_keys = list(ssh_dir.glob("*.pub"))
            if pub_keys:
                self._add_finding(
                    "Identity", "MEDIUM",
                    f"Found {len(pub_keys)} SSH public key(s) in ~/.ssh/",
                    "Use operation-specific SSH keys, not personal keys"
                )

    def audit_shell_history(self):
        """Check shell history for sensitive commands."""
        history_files = [".bash_history", ".zsh_history", ".python_history"]
        for hf in history_files:
            path = Path.home() / hf
            if path.exists():
                size = path.stat().st_size
                if size > 0:
                    self._add_finding(
                        "OpSec", "MEDIUM",
                        f"Shell history exists: ~/{hf} ({size} bytes)",
                        f"Clear history: > ~/{hf} && history -c"
                    )

    def audit_file_metadata(self, target_dir="."):
        """Scan files for metadata that reveals identity."""
        # TODO: Run exiftool on files and flag identity-revealing metadata
        extensions = ("*.jpg", "*.jpeg", "*.png", "*.pdf", "*.docx")
        for ext in extensions:
            for filepath in glob.glob(os.path.join(target_dir, "**", ext), recursive=True):
                try:
                    result = subprocess.run(
                        ["exiftool", "-json", filepath],
                        capture_output=True, text=True, timeout=10
                    )
                    metadata = json.loads(result.stdout)
                    sensitive_fields = ["GPSLatitude", "GPSLongitude", "Author",
                                        "Creator", "Producer", "Company"]
                    for entry in metadata:
                        for field in sensitive_fields:
                            if field in entry:
                                self._add_finding(
                                    "Metadata", "HIGH",
                                    f"Sensitive metadata in {filepath}: {field}={entry[field]}",
                                    f"Remove with: mat2 {filepath}"
                                )
                except (FileNotFoundError, subprocess.TimeoutExpired, json.JSONDecodeError):
                    pass

    def audit_network(self):
        """Check for network configuration leaks."""
        # TODO: Check for DNS leak indicators and exposed services
        try:
            result = subprocess.run(
                ["cat", "/etc/resolv.conf"], capture_output=True, text=True
            )
            if "127.0.0.1" not in result.stdout and "tor" not in result.stdout.lower():
                self._add_finding(
                    "Network", "HIGH",
                    "DNS resolver is not routed through Tor/localhost",
                    "Configure DNS to use 127.0.0.1 when operating through Tor"
                )
        except FileNotFoundError:
            pass

    def run_full_audit(self, target_dir="."):
        """Execute all audit modules."""
        print("[*] GRIMOIRE OpSec Audit Starting...")
        self.audit_git_config()
        self.audit_ssh_keys()
        self.audit_shell_history()
        self.audit_file_metadata(target_dir)
        self.audit_network()
        print(f"[*] Audit complete: {len(self.findings)} findings, risk score: {self.risk_score}")

    def export_report(self, path="opsec_audit.json"):
        """Export the audit report."""
        report = {
            "timestamp": datetime.utcnow().isoformat(),
            "risk_score": self.risk_score,
            "risk_level": "CRITICAL" if self.risk_score > 50 else
                          "HIGH" if self.risk_score > 30 else
                          "MEDIUM" if self.risk_score > 15 else "LOW",
            "total_findings": len(self.findings),
            "findings": self.findings
        }
        with open(path, "w") as f:
            json.dump(report, f, indent=2)
        print(f"[+] Report saved to {path}")
        return report


if __name__ == "__main__":
    import argparse
    parser = argparse.ArgumentParser(description="GRIMOIRE OpSec Auditor")
    parser.add_argument("-d", "--directory", default=".",
                        help="Directory to audit for metadata exposure")
    parser.add_argument("-o", "--output", default="opsec_audit.json",
                        help="Output report path")
    args = parser.parse_args()

    auditor = OpSecAuditor()
    auditor.run_full_audit(args.directory)
    auditor.export_report(args.output)
```
