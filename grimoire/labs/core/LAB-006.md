## 📖 Background

Scripting turns manual processes into repeatable, reliable automation. In cybersecurity, scripting is used for everything from automated scanning to log parsing to incident response playbooks. Bash is essential for system-level automation, while Python provides powerful libraries for complex data processing, network operations, and tool development.

---

## 🔬 Lab Instructions

### Phase 1: Bash Scripting

1. Write a script with variables, string operations, and arithmetic
2. Implement loops (for, while) to process multiple items
3. Use conditionals (if/elif/else) for decision logic
4. Create functions to organize reusable code
5. Process command-line arguments with `$1`, `$@`, and `getopts`

### Phase 2: Python Scripting

1. Write Python scripts with variables, lists, dictionaries, and loops
2. Read and write files programmatically
3. Parse JSON data using the `json` module
4. Make HTTP requests using `urllib` or `requests`
5. Handle errors gracefully with try/except

### Phase 3: Practical Automation

1. Create a Bash script that automates system information gathering
2. Create a Python script that processes log files and extracts patterns
3. Build a script that combines multiple tools into an automated workflow
4. Add command-line arguments, help text, and error handling
