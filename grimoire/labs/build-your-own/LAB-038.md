## 📖 Background

Commercial SIEMs bundle collection, detection, and alerting into opaque products. Understanding each stage independently makes you a far more effective analyst — you can troubleshoot blind spots, write better detection rules, and evaluate vendor claims with precision.

Key concepts:
- **Log normalization** — Mapping disparate log formats to a single schema (timestamp, source, event_type, severity, message, fields)
- **Detection rules** — Declarative conditions that identify suspicious patterns
- **Correlation** — Linking related events across sources and time windows
- **Alert fatigue** — Why deduplication and tuning are essential
- **Pipeline architecture** — Stages connected by queues for decoupled processing

## 🔬 Lab Instructions

### Phase 1: Log Collection and Normalization

1. Create a `LogCollector` class that:
   - Accepts a list of file paths to monitor
   - Tails each file for new lines using a background thread
   - Pushes raw lines into a shared thread-safe queue
2. Create a `LogNormalizer` class that:
   - Reads raw lines from the collection queue
   - Applies format-specific parsers (syslog, JSON, CSV, Apache/Nginx access logs)
   - Outputs normalized event dicts with a common schema:
     `{"timestamp", "source", "event_type", "severity", "message", "fields": {}}`
   - Pushes normalized events into a detection queue
3. Test with sample log files of at least two different formats

### Phase 2: Detection Rule Engine

1. Define a YAML rule schema:
   ```yaml
   - id: RULE-001
     name: Brute Force SSH
     description: Multiple failed SSH logins from the same source
     severity: HIGH
     match:
       field: message
       pattern: "Failed password"
     threshold:
       count: 5
       window_seconds: 60
       group_by: fields.source_ip
   ```
2. Create a `RuleEngine` class that:
   - Loads rules from a YAML file at startup
   - Reads normalized events from the detection queue
   - Evaluates each event against all rules
   - Tracks threshold state using sliding time windows
   - Emits alert dicts when rules trigger
3. Implement both single-event pattern rules and threshold-based rules
4. Test with crafted log data that should trigger your rules

### Phase 3: Alerting and Notification

1. Create an `AlertManager` class that:
   - Receives alert dicts from the rule engine
   - Deduplicates: suppresses repeated alerts for the same rule + group key within a cooldown period
   - Enriches alerts with context (first seen, count, related events)
   - Dispatches to one or more notification channels
2. Implement three notification channels:
   - **Console** — Pretty-printed alert to stdout
   - **File** — Append JSON alert to an alerts log file
   - **Webhook** — POST JSON to a configurable URL (use a local test receiver)
3. Wire all stages together: Collector → Normalizer → RuleEngine → AlertManager
4. Run the full pipeline and inject test log data to verify end-to-end detection

```python
#!/usr/bin/env python3
"""GRIMOIRE Threat Detection Pipeline — LAB-038 Build Your Own"""

import os
import re
import time
import yaml
import json
import threading
from queue import Queue
from datetime import datetime, timedelta
from collections import defaultdict

# --- Shared Queues ---
raw_queue = Queue()
normalized_queue = Queue()
alert_queue = Queue()

# --- Log Collector ---
class LogCollector:
    """Tail one or more log files and push raw lines into the raw queue."""

    def __init__(self, file_paths):
        self.file_paths = file_paths

    def tail_file(self, path):
        """Tail a single log file, yielding new lines."""
        with open(path, "r") as fh:
            fh.seek(0, os.SEEK_END)
            while True:
                line = fh.readline()
                if line:
                    raw_queue.put({"source_file": path, "raw": line.strip()})
                else:
                    time.sleep(0.5)

    def start(self):
        for path in self.file_paths:
            t = threading.Thread(target=self.tail_file, args=(path,), daemon=True)
            t.start()

# --- Log Normalizer ---
class LogNormalizer:
    """Read raw log lines and convert to a common event schema."""

    def normalize(self, raw_entry):
        """Parse a raw log entry into a normalized event dict."""
        # TODO: Implement format detection and field extraction
        # Return: {"timestamp", "source", "event_type", "severity", "message", "fields": {}}
        return {
            "timestamp": datetime.utcnow().isoformat(),
            "source": raw_entry.get("source_file", "unknown"),
            "event_type": "generic",
            "severity": "INFO",
            "message": raw_entry.get("raw", ""),
            "fields": {},
        }

    def run(self):
        while True:
            raw_entry = raw_queue.get()
            event = self.normalize(raw_entry)
            normalized_queue.put(event)

# --- Detection Rule Engine ---
class RuleEngine:
    """Evaluate normalized events against YAML-defined detection rules."""

    def __init__(self, rules_path):
        with open(rules_path, "r") as fh:
            self.rules = yaml.safe_load(fh) or []
        # Threshold tracking: rule_id -> group_key -> list of timestamps
        self.threshold_state = defaultdict(lambda: defaultdict(list))

    def match_pattern(self, event, rule):
        """Check if event matches a rule's pattern condition."""
        field_value = event.get(rule["match"]["field"], "")
        return bool(re.search(rule["match"]["pattern"], field_value))

    def check_threshold(self, event, rule):
        """Check if a threshold rule has been exceeded."""
        # TODO: Implement sliding window threshold logic
        # Track events by group_by key, count within window_seconds
        threshold = rule.get("threshold")
        if not threshold:
            return True  # No threshold — single-event rule
        group_key = event.get("fields", {}).get(
            threshold.get("group_by", "").replace("fields.", ""), "global"
        )
        now = datetime.utcnow()
        window = timedelta(seconds=threshold["window_seconds"])
        state = self.threshold_state[rule["id"]][group_key]
        state.append(now)
        # Prune old entries outside the window
        self.threshold_state[rule["id"]][group_key] = [
            t for t in state if now - t <= window
        ]
        return len(self.threshold_state[rule["id"]][group_key]) >= threshold["count"]

    def evaluate(self, event):
        """Evaluate an event against all rules."""
        for rule in self.rules:
            if self.match_pattern(event, rule) and self.check_threshold(event, rule):
                alert_queue.put({
                    "rule_id": rule["id"],
                    "rule_name": rule["name"],
                    "severity": rule["severity"],
                    "event": event,
                    "triggered_at": datetime.utcnow().isoformat(),
                })

    def run(self):
        while True:
            event = normalized_queue.get()
            self.evaluate(event)

# --- Alert Manager ---
class AlertManager:
    """Deduplicate and dispatch alerts to notification channels."""

    def __init__(self, cooldown_seconds=300):
        self.cooldown = timedelta(seconds=cooldown_seconds)
        self.last_fired = {}  # (rule_id, group_key) -> datetime

    def is_duplicate(self, alert):
        """Check if this alert was already fired within the cooldown window."""
        key = (alert["rule_id"], alert["event"].get("fields", {}).get("source_ip", "global"))
        now = datetime.utcnow()
        if key in self.last_fired and now - self.last_fired[key] < self.cooldown:
            return True
        self.last_fired[key] = now
        return False

    def notify_console(self, alert):
        severity = alert["severity"]
        print(f"[ALERT][{severity}] {alert['rule_name']} — {alert['event']['message']}")

    def notify_file(self, alert, path="alerts.json"):
        with open(path, "a") as fh:
            fh.write(json.dumps(alert, default=str) + "\n")

    def notify_webhook(self, alert, url=None):
        """POST alert JSON to a webhook URL."""
        # TODO: Implement webhook notification using requests.post
        pass

    def run(self):
        while True:
            alert = alert_queue.get()
            if not self.is_duplicate(alert):
                self.notify_console(alert)
                self.notify_file(alert)

# --- Entry Point ---
if __name__ == "__main__":
    import sys

    rules_path = sys.argv[1] if len(sys.argv) > 1 else "rules.yaml"
    log_sources = sys.argv[2:] if len(sys.argv) > 2 else ["/var/log/syslog"]

    collector = LogCollector(log_sources)
    normalizer = LogNormalizer()
    engine = RuleEngine(rules_path)
    manager = AlertManager()

    collector.start()
    for target in [normalizer.run, engine.run, manager.run]:
        threading.Thread(target=target, daemon=True).start()

    print(f"[pipeline] Detection pipeline running — watching {log_sources}")
    try:
        while True:
            time.sleep(1)
    except KeyboardInterrupt:
        print("[pipeline] Shutting down.")
```
