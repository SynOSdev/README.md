## 📖 Background

War games are controlled exercises where a red team attempts to achieve specific objectives (e.g., exfiltrate a file, escalate privileges) while a blue team detects and responds. Tabletop exercises are discussion-based — no live attacks. Live exercises involve actual commands on real or sandboxed systems.

Key concepts:
- **Rules of Engagement (ROE)** — Agreed boundaries defining what attacks are permitted
- **Inject** — A scenario event introduced by the exercise controller to drive action
- **Time-Boxed Round** — A fixed period during which the red team operates before blue team reviews
- **Scoring Rubric** — Criteria for measuring success (detection time, containment speed, objectives achieved)
- **After-Action Review (AAR)** — Structured debrief identifying what worked, what failed, and what to improve

## 🔬 Lab Instructions

### Phase 1: Planning the War Game

1. Define the scenario scope and objectives:
   ```bash
   mkdir -p ~/wargame && cat > ~/wargame/scenario.yaml << 'EOF'
   scenario:
     name: "Operation Shadow Gate"
     type: tabletop  # or "live"
     duration_minutes: 60
     red_team_objective: "Exfiltrate /opt/secrets/flag.txt from target host"
     blue_team_objective: "Detect and contain the intrusion before exfiltration"
     scope:
       in_scope:
         - "10.0.0.0/24 lab network"
         - "Linux hosts only"
       out_of_scope:
         - "Production systems"
         - "Denial of service attacks"
     rules_of_engagement:
       - "No destructive actions"
       - "Red team must log all commands"
       - "Blue team may not block red team IP preemptively"
     injects:
       - time: "T+10m"
         event: "Red team gains initial foothold via SSH brute-force"
       - time: "T+25m"
         event: "Red team escalates privileges using misconfigured sudo"
       - time: "T+40m"
         event: "Red team attempts data exfiltration over DNS"
   EOF
   echo "Scenario created: ~/wargame/scenario.yaml"
   ```
2. Create the scoring rubric:
   ```bash
   cat > ~/wargame/scoring.yaml << 'EOF'
   scoring:
     red_team:
       - objective: "Initial access achieved"
         points: 10
       - objective: "Privilege escalation achieved"
         points: 15
       - objective: "Flag exfiltrated"
         points: 25
     blue_team:
       - objective: "Initial access detected"
         points: 10
       - objective: "Escalation detected within 5 minutes"
         points: 15
       - objective: "Exfiltration blocked"
         points: 25
       - objective: "Incident report filed within 15 minutes"
         points: 10
   EOF
   ```

### Phase 2: Executing the Simulation

1. Set up a split-screen environment for red and blue consoles:
   ```bash
   # Start a tmux session with two panes
   tmux new-session -d -s wargame -n main
   tmux split-window -h -t wargame
   tmux send-keys -t wargame:main.0 'echo "=== RED TEAM CONSOLE ==="' Enter
   tmux send-keys -t wargame:main.1 'echo "=== BLUE TEAM CONSOLE ==="' Enter
   tmux attach -t wargame
   ```
2. Red team actions (left pane — simulated):
   ```bash
   # Reconnaissance
   nmap -sn 10.0.0.0/24 2>/dev/null || echo "[SIM] Network scan: found 5 hosts"
   # Initial access attempt
   echo "[SIM] SSH brute-force against 10.0.0.5 — access gained as user 'deploy'"
   # Privilege escalation
   echo "[SIM] sudo -l reveals NOPASSWD on /usr/bin/vim — escalating to root"
   # Exfiltration attempt
   echo "[SIM] Encoding flag.txt as DNS queries to attacker-controlled domain"
   ```
3. Blue team actions (right pane — simulated):
   ```bash
   # Monitor authentication logs
   echo "[DETECT] Multiple failed SSH attempts from 10.0.0.99 — alert raised"
   # Check for privilege escalation
   echo "[DETECT] Unexpected root shell spawned by user 'deploy' — investigating"
   # Monitor DNS traffic
   echo "[DETECT] Unusual DNS query volume to unknown domain — blocking"
   ```
4. Log all events with timestamps:
   ```bash
   echo "$(date -u +%H:%M:%S) RED  — Initial access achieved" >> ~/wargame/event_log.txt
   echo "$(date -u +%H:%M:%S) BLUE — SSH brute-force detected" >> ~/wargame/event_log.txt
   ```

### Phase 3: Scoring and After-Action Review

1. Calculate scores from the event log:
   ```bash
   red_score=0; blue_score=0
   grep -c "RED.*achieved" ~/wargame/event_log.txt && red_score=$((red_score + 25))
   grep -c "BLUE.*detected" ~/wargame/event_log.txt && blue_score=$((blue_score + 25))
   echo "Red Team: $red_score pts | Blue Team: $blue_score pts"
   ```
2. Conduct the after-action review:
   ```bash
   cat > ~/wargame/aar.md << 'EOF'
   # After-Action Review — Operation Shadow Gate

   ## What Worked
   - Blue team detected SSH brute-force within 2 minutes
   - DNS exfiltration was identified by anomaly detection

   ## What Failed
   - Privilege escalation went undetected for 8 minutes
   - No automated containment was triggered

   ## Improvements
   - Add detection rule for unexpected sudo usage
   - Implement automated account lockout on brute-force detection
   - Reduce mean-time-to-contain (MTTC) target to 5 minutes
   EOF
   echo "AAR saved: ~/wargame/aar.md"
   ```

## 🛠️ Build Your Own

**Build Objective:** Create a custom war game scenario generator and scoring engine

### Requirements
- Generate randomized war game scenarios from configurable templates
- Define red and blue team objectives with point values
- Track events in real time and calculate scores
- Produce a post-exercise report with metrics

### Starter Framework
```python
#!/usr/bin/env python3
"""GRIMOIRE War Game Engine — LAB-027 Build Your Own"""

import json
import random
import os
from datetime import datetime, timezone

SCENARIOS_DIR = os.path.expanduser("~/wargame/scenarios")
os.makedirs(SCENARIOS_DIR, exist_ok=True)

ATTACK_VECTORS = [
    "SSH brute-force", "Phishing payload delivery", "Exploit public-facing app",
    "Supply chain compromise", "Credential stuffing", "Insider threat simulation",
]
ESCALATION_METHODS = [
    "Misconfigured sudo", "Kernel exploit", "SUID binary abuse",
    "Token impersonation", "Password reuse from credential dump",
]
EXFIL_METHODS = [
    "DNS tunneling", "HTTPS to cloud storage", "ICMP covert channel",
    "Steganography in image upload", "Encrypted archive to external drive",
]

def generate_scenario(name: str) -> dict:
    """Build a randomized war game scenario."""
    return {
        "name": name,
        "generated_at": datetime.now(timezone.utc).isoformat(),
        "duration_minutes": random.choice([30, 45, 60, 90]),
        "attack_vector": random.choice(ATTACK_VECTORS),
        "escalation_method": random.choice(ESCALATION_METHODS),
        "exfil_method": random.choice(EXFIL_METHODS),
        "red_objectives": [
            {"objective": "Gain initial access", "points": 10},
            {"objective": "Escalate to root/admin", "points": 15},
            {"objective": "Exfiltrate target data", "points": 25},
        ],
        "blue_objectives": [
            {"objective": "Detect initial access", "points": 10},
            {"objective": "Detect escalation within 5 min", "points": 15},
            {"objective": "Block exfiltration", "points": 25},
            {"objective": "Complete incident report", "points": 10},
        ],
        "events": [],
    }

def log_event(scenario: dict, team: str, description: str, points: int = 0):
    """Record a timestamped event."""
    scenario["events"].append({
        "timestamp": datetime.now(timezone.utc).isoformat(),
        "team": team,
        "description": description,
        "points": points,
    })
    print(f"  [{team.upper()}] {description} (+{points} pts)")

def score_game(scenario: dict) -> dict:
    """Calculate final scores."""
    red = sum(e["points"] for e in scenario["events"] if e["team"] == "red")
    blue = sum(e["points"] for e in scenario["events"] if e["team"] == "blue")
    return {"red_score": red, "blue_score": blue, "winner": "red" if red > blue else "blue"}

def generate_report(scenario: dict, scores: dict) -> str:
    """Produce a markdown report."""
    lines = [
        f"# War Game Report: {scenario['name']}",
        f"**Date:** {scenario['generated_at']}",
        f"**Duration:** {scenario['duration_minutes']} minutes",
        f"**Attack Vector:** {scenario['attack_vector']}",
        f"**Escalation:** {scenario['escalation_method']}",
        f"**Exfiltration:** {scenario['exfil_method']}",
        "",
        "## Scores",
        f"- Red Team: {scores['red_score']} pts",
        f"- Blue Team: {scores['blue_score']} pts",
        f"- **Winner: {scores['winner'].upper()} TEAM**",
        "",
        "## Event Log",
    ]
    for e in scenario["events"]:
        lines.append(f"- `{e['timestamp']}` [{e['team'].upper()}] {e['description']} (+{e['points']})")
    return "\n".join(lines)

if __name__ == "__main__":
    sc = generate_scenario("Operation Shadow Gate")
    print(f"=== Scenario: {sc['name']} ===")
    print(f"Attack: {sc['attack_vector']} → {sc['escalation_method']} → {sc['exfil_method']}")
    print(f"Duration: {sc['duration_minutes']} minutes\n")

    # Simulate events
    log_event(sc, "red", "Initial access via " + sc["attack_vector"], 10)
    log_event(sc, "blue", "Detected initial access attempt", 10)
    log_event(sc, "red", "Escalated using " + sc["escalation_method"], 15)
    log_event(sc, "blue", "Escalation detected after 7 minutes", 10)
    log_event(sc, "red", "Exfiltrated data via " + sc["exfil_method"], 25)
    log_event(sc, "blue", "Exfiltration partially blocked", 15)

    scores = score_game(sc)
    report = generate_report(sc, scores)
    report_path = os.path.join(SCENARIOS_DIR, f"{sc['name'].replace(' ', '_')}.md")
    with open(report_path, "w") as f:
        f.write(report)
    print(f"\n[+] Report saved: {report_path}")
    print(f"Final Score — Red: {scores['red_score']} | Blue: {scores['blue_score']} | Winner: {scores['winner'].upper()}")
```

### Validation
- [ ] Scenario generator produces randomized but valid scenarios
- [ ] Events are logged with timestamps and point values
- [ ] Scoring engine calculates correct totals for both teams
- [ ] Markdown report is generated with all exercise details
