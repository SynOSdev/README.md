## 📖 Background

Detection engineering is the discipline of building, testing, and maintaining detection rules at scale. Modern detection teams treat rules as code — version-controlled, tested in CI pipelines, and deployed through automation. The three major rule formats cover complementary domains: Sigma for logs, YARA for files, and Snort/Suricata for network traffic.

Key concepts:
- **Sigma Rule** — A YAML-based detection rule that can be compiled to any SIEM query language
- **YARA Rule** — A pattern-matching rule for identifying files by byte patterns, strings, or conditions
- **Snort/Suricata Rule** — A network detection rule matching packet headers and payloads
- **True Positive Rate** — The percentage of real attacks correctly identified by a rule
- **Detection-as-Code** — Managing detection rules with version control, testing, and automated deployment

### Phase 1: Writing Sigma Rules

1. Create a Sigma rule for detecting encoded PowerShell/Bash execution (T1059.001):
   ```bash
   mkdir -p ~/detection_rules/sigma
   cat > ~/detection_rules/sigma/encoded_shell_execution.yml << 'EOF'
   title: Encoded Shell Command Execution
   id: d4f7a8b2-1234-5678-9abc-def012345678
   status: experimental
   description: Detects execution of shell commands with base64-encoded arguments
   references:
     - https://attack.mitre.org/techniques/T1059/001/
   author: GRIMOIRE Lab
   date: 2024/01/15
   tags:
     - attack.execution
     - attack.t1059.001
   logsource:
     category: process_creation
     product: linux
   detection:
     selection_binary:
       Image|endswith:
         - '/bash'
         - '/sh'
         - '/powershell'
     selection_encoded:
       CommandLine|contains:
         - '-enc '
         - '-EncodedCommand'
         - 'base64 -d'
         - 'base64 --decode'
     condition: selection_binary and selection_encoded
   falsepositives:
     - Legitimate admin scripts using base64 encoding
     - CI/CD pipelines with encoded configuration
   level: medium
   EOF
   echo "Sigma rule created: encoded_shell_execution.yml"
   ```
2. Create a second Sigma rule for scheduled task persistence (T1053.005):

   ```bash
   description: Detects creation of scheduled tasks or cron jobs by non-root users
   references:
     - https://attack.mitre.org/techniques/T1053/005/
   ```

### Phase 2: Writing YARA Rules

1. Create a YARA rule that detects a simulated malware dropper pattern:
   ```bash
   mkdir -p ~/detection_rules/yara
   cat > ~/detection_rules/yara/suspicious_dropper.yar << 'EOF'
   rule GRIMOIRE_Suspicious_Dropper
   {
       meta:
           description = "Detects files with characteristics of a malware dropper"
           author = "GRIMOIRE Lab"
           date = "2024-01-15"
           reference = "https://attack.mitre.org/techniques/T1204/002/"
           severity = "high"

       strings:
           $download_func = "urllib.request.urlretrieve" ascii
           $wget_call = /wget\s+https?:\/\// ascii
           $curl_call = /curl\s+-[sSkLo]+\s+https?:\/\// ascii
           $exec_payload = /exec\(|eval\(|os\.system\(/ ascii
           $temp_path = /\/tmp\/[a-zA-Z0-9]{6,}/ ascii
           $base64_decode = "base64.b64decode" ascii

       condition:
           filesize < 50KB and
           (($download_func or $wget_call or $curl_call) and
            ($exec_payload or $base64_decode) and
            $temp_path)
   }
   EOF
   echo "YARA rule created: suspicious_dropper.yar"
   ```
2. Test the YARA rule against a sample file:
   ```bash
   # Create a benign test file that should NOT match
   echo "Hello world" > /tmp/benign.txt
   # Create a suspicious test file that SHOULD match
   cat > /tmp/suspicious.py << 'TESTEOF'
   import urllib.request
   import base64
   data = base64.b64decode("dGVzdA==")
   exec(data)
   urllib.request.urlretrieve("http://example.com", "/tmp/abcdef123456")
   TESTEOF
   yara ~/detection_rules/yara/suspicious_dropper.yar /tmp/suspicious.py 2>/dev/null \
     && echo "[PASS] YARA detected suspicious file" \
     || echo "[INFO] Install yara to test: apt install yara"
   rm -f /tmp/benign.txt /tmp/suspicious.py
   ```

### Phase 3: Writing Suricata Rules

1. Create a Suricata rule for detecting DNS exfiltration patterns:
   ```bash

   cat > ~/detection_rules/suricata/dns_exfil.rules << 'EOF'

   alert dns any any -> any any (msg:"GRIMOIRE - Possible DNS Tunneling - Long Query"; \
     dns.query; content:"."; pcre:"/^[a-zA-Z0-9\-]{40,}\./"; \

     classtype:bad-unknown; sid:9000001; rev:1;)
   ```

## 🛠️ Build Your Own

**Build Objective:** Create a custom detection rule management system that stores, tests, and deploys detection rules

### Requirements
- Store Sigma, YARA, and Suricata rules in a structured directory
- Parse and validate rules for syntax correctness
- Test rules against sample data and report true/false positive rates
- Track rule metadata (author, date, ATT&CK mapping, status)
- Generate a coverage report showing which ATT&CK techniques have detection rules

### Starter Framework
```python
#!/usr/bin/env python3
"""GRIMOIRE Detection Rule Manager — LAB-030 Build Your Own"""

import json
import os
import re
import sys
from datetime import datetime, timezone
from pathlib import Path

RULES_DIR = os.path.expanduser("~/detection_rules")
CATALOG_FILE = os.path.join(RULES_DIR, "catalog.json")

def scan_rules() -> list:
    """Scan the rules directory and build a catalog."""
    catalog = []
    rules_path = Path(RULES_DIR)
    for rule_file in rules_path.rglob("*"):
        if rule_file.suffix in (".yml", ".yaml"):
            catalog.append(parse_sigma_rule(rule_file))
        elif rule_file.suffix == ".yar":
            catalog.append(parse_yara_rule(rule_file))
        elif rule_file.suffix == ".rules":
            catalog.extend(parse_suricata_rules(rule_file))
    return [c for c in catalog if c]

def parse_sigma_rule(path: Path) -> dict:
    """Extract metadata from a Sigma rule file."""
    content = path.read_text()
    title = ""
    level = ""
    tags = []
    for line in content.splitlines():
        if line.startswith("title:"):
            title = line.split(":", 1)[1].strip()
        elif line.startswith("level:"):
            level = line.split(":", 1)[1].strip()
        elif line.strip().startswith("- attack.t"):
            tags.append(line.strip().lstrip("- "))
    techniques = [t.replace("attack.", "").upper().replace(".", "")
                  for t in tags if t.startswith("attack.t")]
    return {
        "file": str(path),
        "type": "sigma",
        "title": title,
        "level": level,
        "techniques": techniques,
    }

def parse_yara_rule(path: Path) -> dict:
    """Extract metadata from a YARA rule file."""
    content = path.read_text()
    rule_name = ""
    match = re.search(r"rule\s+(\w+)", content)
    if match:
        rule_name = match.group(1)
    ref_match = re.search(r'reference\s*=\s*"([^"]*)"', content)
    technique = ""
    if ref_match and "T" in ref_match.group(1):
        tid = re.search(r"T\d{4}(?:/\d{3})?", ref_match.group(1))
        if tid:
            technique = tid.group(0)
    return {
        "file": str(path),
        "type": "yara",
        "title": rule_name,
        "level": "high",
        "techniques": [technique] if technique else [],
    }

def parse_suricata_rules(path: Path) -> list:
    """Extract metadata from a Suricata rules file."""
    entries = []
    for line in path.read_text().splitlines():
        if line.startswith("#") or not line.strip():
            continue
        msg_match = re.search(r'msg:"([^"]+)"', line)
        sid_match = re.search(r'sid:(\d+)', line)
        if msg_match:
            entries.append({
                "file": str(path),
                "type": "suricata",
                "title": msg_match.group(1),
                "sid": sid_match.group(1) if sid_match else "",
                "level": "medium",
                "techniques": [],
            })
    return entries

def test_sigma_against_logs(rule_path: str, logs_path: str):
    """Simple matcher: check if log entries match Sigma rule conditions."""
    content = Path(rule_path).read_text()
    # Extract detection keywords (simplified)
    keywords = []
    in_detection = False
    for line in content.splitlines():
        if line.startswith("detection:"):
            in_detection = True
            continue
        if in_detection and line.strip().startswith("- '"):
            kw = line.strip().strip("- '\"")
            keywords.append(kw)
        if in_detection and not line.startswith(" ") and line.strip() and not line.startswith("detection"):
            in_detection = False

    with open(logs_path) as f:
        logs = json.load(f)

    matches = 0
    for entry in logs:
        cmd = entry.get("CommandLine", "")
        if any(kw in cmd for kw in keywords):
            matches += 1
    total = len(logs)
    print(f"  Rule: {os.path.basename(rule_path)}")
    print(f"  Matches: {matches}/{total} log entries")
    print(f"  Match rate: {matches*100//total if total else 0}%")

def coverage_report(catalog: list):
    """Show ATT&CK technique coverage from all rules."""
    all_techniques = set()
    for entry in catalog:
        all_techniques.update(entry.get("techniques", []))
    print(f"\n{'='*50}")
    print(f"  Detection Rule Coverage Report")
    print(f"  Total rules: {len(catalog)}")
    print(f"  ATT&CK techniques covered: {len(all_techniques)}")
    print(f"{'='*50}")
    print(f"\n  {'Type':<12} {'Count':>6}")
    print(f"  {'-'*20}")
    for rtype in ("sigma", "yara", "suricata"):
        count = sum(1 for c in catalog if c["type"] == rtype)
        print(f"  {rtype:<12} {count:>6}")
    if all_techniques:
        print(f"\n  Techniques with detections:")
        for t in sorted(all_techniques):
            print(f"    - {t}")

if __name__ == "__main__":
    if len(sys.argv) < 2:
        print("Usage: manager.py {scan|test|coverage}")
        sys.exit(1)
    cmd = sys.argv[1]
    if cmd == "scan":
        catalog = scan_rules()
        with open(CATALOG_FILE, "w") as f:
            json.dump(catalog, f, indent=2)
        print(f"[+] Cataloged {len(catalog)} rules -> {CATALOG_FILE}")
    elif cmd == "test" and len(sys.argv) > 3:
        test_sigma_against_logs(sys.argv[2], sys.argv[3])
    elif cmd == "coverage":
        if os.path.exists(CATALOG_FILE):
            with open(CATALOG_FILE) as f:
                catalog = json.load(f)
            coverage_report(catalog)
        else:
            print("Run 'scan' first to build the catalog.")
    else:
        print("Usage: manager.py {scan|test <rule> <logs>|coverage}")
```
