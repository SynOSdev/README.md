## 📖 Background

Purple teaming bridges the gap between red (attack) and blue (defend) by running attacks in a controlled way and immediately measuring the defensive response. The key insight is that untested detections are unreliable — you must validate them with real attack telemetry.

Key concepts:
- **Atomic Test** — A small, self-contained attack simulation that exercises one ATT&CK technique
- **Detection Coverage** — The percentage of ATT&CK techniques your detections can identify
- **Coverage Gap** — A technique for which no detection rule exists or fires
- **Attack-Defense Alignment** — Mapping every offensive action to a corresponding defensive control
- **True Positive Validation** — Confirming that a detection fires correctly when the attack runs

## 🔬 Lab Instructions

### Phase 1: Setting Up the Attack Simulation Environment

1. Clone the Atomic Red Team repository:
   ```bash
   git clone https://github.com/redcanaryco/atomic-red-team.git ~/art
   cd ~/art
   ```
2. List available techniques and pick five from different tactics:
   ```bash
   ls atomics/ | head -20
   # Example picks: T1059.004 (Execution), T1053.003 (Persistence),
   # T1070.004 (Defense Evasion), T1018 (Discovery), T1041 (Exfiltration)
   ```
3. Review the YAML definition for your first test:
   ```bash
   cat atomics/T1059.004/T1059.004.yaml | head -40
   ```
4. Ensure your SIEM or log collector is running and ingesting endpoint logs (syslog, auditd, or osquery).

### Phase 2: Executing Atomic Tests and Collecting Telemetry

1. Run an atomic test manually following the YAML instructions:
   ```bash
   # Example: T1059.004 — Unix Shell command execution
   bash -c 'echo "GRIMOIRE atomic test T1059.004 — $(date)"' | tee /tmp/art_test.log
   ```
2. For each test, record the timestamp, technique ID, and expected log artifacts:
   ```bash
   echo "$(date -u +%Y-%m-%dT%H:%M:%SZ),T1059.004,process_creation,bash" >> ~/art_results.csv
   ```
3. Check your SIEM for corresponding alerts within 5 minutes of each test:
   ```bash
   # Example: search syslog for evidence
   grep -i "atomic test" /var/log/syslog 2>/dev/null || echo "Check your SIEM UI"
   ```
4. Repeat for all five selected techniques, logging results each time.

### Phase 3: Building the Detection Coverage Map

1. Create a JSON coverage file from your results:
   ```bash
   cat ~/art_results.csv | while IFS=',' read -r ts tid source cmd; do
     detected="false"
     # Set detected=true if your SIEM fired an alert for this technique
     echo "{\"technique\":\"$tid\",\"detected\":$detected,\"timestamp\":\"$ts\"}"
   done | jq -s '.' > ~/coverage_map.json
   ```
2. Summarize coverage:
   ```bash
   total=$(jq length ~/coverage_map.json)
   detected=$(jq '[.[] | select(.detected==true)] | length' ~/coverage_map.json)
   echo "Coverage: $detected / $total techniques detected ($(( detected * 100 / total ))%)"
   ```
3. Export the results in ATT&CK Navigator layer format for visualization:
   ```bash
   jq '[.[] | {techniqueID: .technique, score: (if .detected then 1 else 0 end), color: (if .detected then "#00ff00" else "#ff0000" end)}]' \
     ~/coverage_map.json > ~/navigator_layer_techniques.json
   ```
4. Open ATT&CK Navigator (https://mitre-attack.github.io/attack-navigator/) and import your layer to see red (gap) and green (covered) techniques.

## 🛠️ Build Your Own

**Build Objective:** Create a custom attack-to-detection mapper that runs tests and checks if detections fire

### Requirements
- Accept a list of ATT&CK technique IDs as input
- Execute a simulated test action for each technique
- Query a log source to determine if a detection fired
- Produce a coverage report with pass/fail per technique
- Output an ATT&CK Navigator-compatible JSON layer

### Starter Framework
```python
#!/usr/bin/env python3
"""GRIMOIRE Attack-to-Detection Mapper — LAB-025 Build Your Own"""

import json
import subprocess
import time
import os
from datetime import datetime, timezone

LOG_SOURCE = "/var/log/syslog"
RESULTS_FILE = os.path.expanduser("~/detection_coverage.json")

# Simulated atomic actions keyed by technique ID
ATOMIC_ACTIONS = {
    "T1059.004": "bash -c 'echo GRIMOIRE_ART_T1059_004'",
    "T1053.003": "echo '* * * * * echo test' > /tmp/art_cron_test",
    "T1070.004": "touch /tmp/art_indicator && rm /tmp/art_indicator",
    "T1018": "hostname; whoami; ip addr show",
    "T1041": "curl -s --max-time 2 http://example.com -o /dev/null || true",
}

def run_atomic(technique_id: str) -> dict:
    """Execute a simulated atomic test and return metadata."""
    cmd = ATOMIC_ACTIONS.get(technique_id)
    if not cmd:
        return {"technique": technique_id, "status": "no_test", "timestamp": ""}
    ts = datetime.now(timezone.utc).isoformat()
    try:
        subprocess.run(cmd, shell=True, timeout=10, capture_output=True)
    except subprocess.TimeoutExpired:
        pass
    return {"technique": technique_id, "status": "executed", "timestamp": ts}

def check_detection(technique_id: str, after_ts: str) -> bool:
    """Check if a detection fired for the given technique after the timestamp."""
    # TODO: Replace with real SIEM API query or log search
    marker = f"GRIMOIRE_ART_{technique_id.replace('.', '_')}"
    try:
        result = subprocess.run(
            ["grep", "-c", marker, LOG_SOURCE],
            capture_output=True, text=True, timeout=5,
        )
        return int(result.stdout.strip()) > 0
    except (subprocess.TimeoutExpired, ValueError):
        return False

def generate_navigator_layer(results: list) -> dict:
    """Produce an ATT&CK Navigator layer JSON."""
    techniques = []
    for r in results:
        techniques.append({
            "techniqueID": r["technique"],
            "score": 1 if r["detected"] else 0,
            "color": "#00ff00" if r["detected"] else "#ff0000",
        })
    return {
        "name": "GRIMOIRE Detection Coverage",
        "versions": {"attack": "14", "navigator": "4.9", "layer": "4.5"},
        "domain": "enterprise-attack",
        "techniques": techniques,
    }

def main(technique_ids: list):
    results = []
    for tid in technique_ids:
        print(f"[*] Running atomic test for {tid}...")
        meta = run_atomic(tid)
        time.sleep(2)
        detected = check_detection(tid, meta["timestamp"])
        meta["detected"] = detected
        results.append(meta)
        status = "DETECTED" if detected else "MISSED"
        print(f"    [{status}] {tid}")

    # Coverage summary
    total = len(results)
    det = sum(1 for r in results if r["detected"])
    print(f"\n=== Coverage: {det}/{total} ({det*100//total if total else 0}%) ===")

    # Save results
    with open(RESULTS_FILE, "w") as f:
        json.dump(results, f, indent=2)

    # Save Navigator layer
    layer = generate_navigator_layer(results)
    layer_path = os.path.expanduser("~/navigator_layer.json")
    with open(layer_path, "w") as f:
        json.dump(layer, f, indent=2)
    print(f"[+] Navigator layer saved to {layer_path}")

if __name__ == "__main__":
    main(list(ATOMIC_ACTIONS.keys()))
```
