# 🎭 GRIMOIRE Roles

Select your role during the game intro. Your role determines your educational specialization within your faction.

---

## 🔍 Analyst

> *"Data tells the story. I read between the lines."*

**Focus:** Security monitoring, log analysis, alert triage, and pattern recognition.

**Core Skills:**
- SIEM operation and query writing
- Log correlation and anomaly detection
- Alert investigation and escalation
- Threat indicator analysis
- Report generation and documentation

**synOS Build Focus:** Custom dashboards, automated alert pipelines, analysis workbenches

**Best Paired With:** Azure Shield, Violet Nexus

---

## ⚙️ Engineer

> *"If it can be built, it can be built better. If it can be automated, it should be."*

**Focus:** Building security tools, automation, infrastructure hardening, and DevSecOps.

**Core Skills:**
- Security tool development
- Infrastructure as Code
- CI/CD pipeline security
- Automation scripting
- System hardening and configuration

**synOS Build Focus:** Custom tool development, automated security pipelines, hardened configurations

**Best Paired With:** Any faction — Engineers are universal force multipliers

---

## 🎯 Hunter

> *"They think they're hidden. They're wrong."*

**Focus:** Proactive threat hunting, adversary tracking, and advanced detection.

**Core Skills:**
- Hypothesis-driven hunting
- Behavioral analysis
- Advanced query crafting
- Threat actor TTPs
- IOC development and enrichment

**synOS Build Focus:** Hunting workstations, custom detection rules, threat feeds

**Best Paired With:** Crimson Circuit, Shadow Protocol

---

## 🏗️ Architect

> *"Security isn't a feature. It's the foundation."*

**Focus:** Security system design, network architecture, policy development, and strategic planning.

**Core Skills:**
- Network security architecture
- Zero-trust design principles
- Security policy frameworks
- Risk assessment and management
- Compliance and governance

**synOS Build Focus:** Network topology tools, policy engines, architecture visualization

**Best Paired With:** Azure Shield, Violet Nexus

---

## Role Selection

Your role affects:
- **Skill emphasis** — Which aspects of each lab you focus on
- **Build priorities** — Role-aligned components in your synOS setup
- **Side quest availability** — Role-specific bonus challenges
- **Personal guides** — Tailored documentation based on your specialization
- **Post-game report** — Skills assessment aligned with your role

> **Note:** Your role shapes *how* you approach labs, while your faction shapes *which* labs you prioritize.
